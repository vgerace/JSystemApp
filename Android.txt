Arquivos principais:

Dentro da pasta arvore, basicamente 3 arquivos:

manifests - um arquivo só, arquivo
de configuração que é o androidmanifest.xml.
Uma das classes mais importantes é a Activity,
e toda criada deve estar registrada neste arquivo. (Aqui no meu pc está dentro de main)

<action android:name="android.intent.action.MAIN" /> 

(Informa que é a Activity principal e quando o app abrir vai abrir estela tela que está relacionada com a Activity)


<category android:name="android.intent.category.LAUNCHER" />

(Quando fazer a instalação do aplicativo é criado um ícone do app)

java

Todos os arquivos kotlin deverão estar dentro desta pasta,
aqui que ficam as classes, etc. Sempre no nome do pacote principal que não tem nada entre parenteses

res

Recursos do projeto, todo resto... se não for manifest, kotlin e dependencias, estará aqui
O que entra em recurso? imagem -> dentro de drawable ou mipmap (mas normalmente é só
para ficar mais um pouco separado de img de icone do app), layout (arquivo de tela, é um
xml, config de tela do aplicativo)

Gradle Scripts

São os arquivos que são gerenciados as dependencias do projeto,
quais bibliotecas podemos ter no projeto que não estão por padrão no sdk.
